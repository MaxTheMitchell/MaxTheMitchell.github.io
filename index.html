The Sight of Music

The "Sight of Music" is a project I initially had the idea for while working at Sycamore-Semi where I was working on pulling data from web API's using a micro-controller to display the status of different Fab equipment. Working on this birthed the idea in me that I could use Spotify's Web API to get the album cover of my currently playing song and use that data to create some sort of display. The first thing I did was to create a website that would allow a user to authorize the website to read their Spotify data and to then parse this data into a bitmap for a micro-controller to use. Next I stated working on the display. The first thing I used to make the display was a strip of 300 led lights that I cut up and sodered into a 16x16 grid. I then created a program for an esp8266 that would issue get requests to the website I made and uses the bitmap that it recieved to make the display. This method worked pretty well for simple album covers, but for album covers that were more complex a 16x16 grid wouldn't cut it. So then I ordered a 64x64 led matrix and upgraded to an esp32 because I needed a better CPU. I was really happy with the results of the 64x64 matrix.
